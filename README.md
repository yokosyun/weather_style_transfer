# Neural Style Transfer for weather editing


## Install
```
python3.8 -m venv ~/venv/nst
source ~/venv/nst/bin/activate
pip install pip --upgrade
pip install -r requirements.txt
pre-commit install
```

## Run NST
```
python neural_style_transfer.py
```
